# himanshu-portfolio
This isn’t your average developer portfolio. This is a full operating system clone (yes, like actual XP) made from scratch in React. It’s my way of saying:

“Hey, I’m a developer. But I’m also weird, creative, nostalgic, and lowkey cracked.”

 Features:

XP boot animation

Login screen with fake password

Draggable windows

Fake apps (browser, terminal, music, notepad, etc.)

Widgets (weather, clock)

Fully working taskbar and Start menu

XP sound effects

Easter eggs 

Mobile-friendly layout

It’s chaotic. It’s unnecessary.
It’s me. 

🛠 Built With
React + Vite – modern stack for retro vibes

TailwindCSS – utility-first styling

react-rnd – for draggable windows

Framer Motion – smooth XP-style animations

XP sounds + icons – nostalgia.exe

Vercel – deployed for free, forever
